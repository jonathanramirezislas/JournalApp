## üõ† Installation
1. Clone this project.
```bash
  gh repo clone jonathanramirezislas/JournalApp
```
2. Change of directory to the root of the project.
```bash
  cd JournalApp
```
3. Install the dependencies.
```bash
  npm install
```
4. Run it locally
```bash
  npm run dev
```
